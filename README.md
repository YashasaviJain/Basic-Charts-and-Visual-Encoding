## 📘 Theory

Data visualization is a key component of data analysis that helps in presenting data in a graphical or pictorial format. It allows users to understand patterns, trends, and relationships in data more easily compared to raw numerical data.

### 🔹 1. Line Chart (Trend Representation)
A line chart is used to display data points connected by straight lines. It is especially useful for showing trends over time or continuous data.

- Helps track changes over intervals (e.g., days, months).
- Multiple lines can be used to compare different variables.
- Markers and legends enhance readability.

### 🔹 2. Bar Chart (Comparison)
A bar chart represents categorical data with rectangular bars. The height or length of each bar corresponds to the value it represents.

- Useful for comparing quantities across categories.
- Can be simple or grouped (multiple variables).
- Adding labels and grids improves clarity.

### 🔹 3. Histogram (Distribution)
A histogram shows the frequency distribution of numerical data by grouping values into bins.

- Helps understand data spread and central tendency.
- Displays skewness and distribution patterns.
- Mean values can be highlighted for analysis.

### 🔹 4. Scatter Plot (Relationship)
A scatter plot displays values for two variables as points on a graph.

- Used to identify relationships or correlations.
- Patterns indicate positive, negative, or no correlation.
- Can include color coding for categorical differentiation.

### 🔹 5. Advanced Visualization with Seaborn
Seaborn is a higher-level visualization library built on top of matplotlib.

- Provides more attractive and informative statistical graphics.
- Simplifies complex visualizations.
- Supports themes and built-in datasets.

### 🔹 Data Encoding Concepts
Visual encoding refers to how data is represented visually using:

- **Position** (x and y axes)
- **Color** (categorical differentiation)
- **Size** (magnitude representation)
- **Shape** (classification)

Effective encoding ensures that the visualization communicates information clearly and efficiently.
